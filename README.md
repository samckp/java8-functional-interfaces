# java8-functional-interfaces

Predefined Functional Interfaces : 
    - Predicate
    - Function
    - Consumer 
    - Supplier

Two argument Predicate Functional Interface
    - BiPredicate
    - BiFunctio
    - BiConsumer

Primitive Functional Interface
    - IntPredicate
    - IntFunction 
    - IntConsumer
-- These are available on java.util.function packge.
